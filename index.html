Tentang WorldStar.ai

WorldStar.ai adalah antarmuka chat AI sederhana yang dibangun menggunakan teknologi Generative AI dari OpenRouter.ai (menggunakan model openai/gpt-5.1-chat).

Tujuan Aplikasi:

  • Menyediakan antarmuka chat yang cepat dan bersih.
  • Menggunakan server proxy PHP (`chat.php`) untuk **keamanan API Key**.
  • Menyimpan riwayat chat secara lokal di *browser* Anda.

Teknologi yang Digunakan:

Frontend: HTML, CSS, JavaScript

Backend: PHP (dengan cURL) sebagai proxy API

WorldStar.ai dapat membuat kesalahan. Pertimbangkan untuk memverifikasi informasi penting.
